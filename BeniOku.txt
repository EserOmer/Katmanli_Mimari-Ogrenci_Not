	Hocam, projemi sql veri tabanı kullanarak yaptım. Bağlantı konusu dediğiniz gibi çok karmaşıtı fakat uzun uğraşlar sonucu
çözümü sağladım. Projede veri tabanıdaki bilgilere erişmek için proje dosyasında olan veri tabanını kendi bilgisayarınızda
tanımlamanız gerekmektedir.

	Benim kullandığım sql, sql express 2019. Sizdeki muhtemelen 2012 versiyonudur. Bunun için benim Database içinde bulunan
veri tabanımı kendi bilgisayarınıza eklemek istediğinizde hata alacaksınız. Buna çözüm olarak benim yapmış olduğum veri tabanını
sıfırdan kendi bilgisayarınızda yazmak olacaktır.

	Tam olarak hangi sürümü kullandığınızı bilmediğim için bütüm sürümler için bu veri tabanı oluşturma kodlarını çıkarttım.
Sizin yapmanız gerekn ise Database içerisinde yer alan, bu oluşturmuş olduğum querylerden size uygun olanını seçip, veri tabanınıza 
DBTESTKATMAN isimli bir veri tabanı oluşturduktan sonra bu seçmiş olduğunuz queryi çalıştırmak. Mevcut sql server sürümünüz 
için gerekli sql query si yok ise çok kısa bir zamanda size ulaştırma işleminni yapabilirim.
	
	Query lerin içinde örnek veriler de mevcuttur.
